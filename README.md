# calculadora-js
Construcción de una calculadora científica paso a paso. 

Este repositorio forma parte de un desafío personal: construir una calculadora científica desde cero en JavaScript, implementando cada cálculo desde la matemática y el código, sin usar librerías externas.

## Primer paso: Simplificación de fracciones
- Algoritmo implementado: Euclides para MCD
- Aplicación: simplificación de fracciones de cualquier tipo

## Próximos pasos
- Suma, resta, multiplicación y división de fracciones
- Operaciones con exponentes y raíces
- Construcción progresiva de la calculadora científica completa
